Utilities

Word Tagger

Import a list of words. Create some tags. Serves up 1 word at a time to be tagged manually. Also includes a picker to pick a random word from the list with a selected tag. Needs a web server with MySQL and PHP to run.


Word Inventor

Very simple random word generator. I use this for video game character names mostly.
